## 升级步骤：

1. 下载 firmware /Atmosphere / hekate 最新版，解压并复制
2. 保留根目录的 `/bootloader/payloads/fusee.bin`
3. 更新 [dbi](https://github.com/rashevskyv/dbi/releases)
4. 短接器接到右侧手柄，破解器插上，按住音量+，点一下电源开机，选择进入破解系统
5. 进入相册-Daybreak，
